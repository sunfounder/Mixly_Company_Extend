**17Maker arduino库-mixly版**
=========
17Maker libary for mixly.
----
The port is general GVS.
Author :hznupeter
E-mail:qiujiongtao@163.com
website: http://hznupeter.oschina.io/maker17/
--
使用方法
1.打开mixly 软件，点击导入库。
2.选中17maker中的maker17.xml文件，稍等片刻，提示导入成功。
